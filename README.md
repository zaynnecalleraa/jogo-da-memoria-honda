# Jogo da Memória — Segurança do Trabalho | HONDA

> Sistema web gamificado de treinamento em segurança do trabalho, desenvolvido com foco em engajamento e aprendizado interativo para a **HONDA**.

---

## 🏢 Sobre a HONDA

A **HONDA** é uma das maiores empresas do setor automotivo e industrial do mundo, reconhecida pela excelência em inovação, qualidade e, principalmente, **segurança no ambiente de trabalho**.

Este projeto foi pensado para contribuir com treinamentos internos, utilizando tecnologia e gamificação para melhorar a retenção de conhecimento dos colaboradores.

---

## 💡 Sobre o Projeto

O **Jogo da Memória — Segurança do Trabalho** é uma aplicação web que transforma conteúdos de segurança em uma experiência interativa.

Diferente de treinamentos tradicionais, o sistema utiliza:
- Memória visual
- Tempo de reação
- Tomada de decisão

👉 Isso torna o aprendizado mais eficiente e envolvente.

---

## 🎮 Como funciona

O jogo possui dois modos:

### 🃏 Modo Memória
O jogador deve encontrar todos os pares de cartas no menor tempo possível.

### 🧠 Modo Memória + Quiz
Ao acertar um par:
- Uma pergunta de segurança é exibida
- O jogador precisa responder corretamente para continuar

---

## ⏱️ Condição de término

O jogo finaliza quando:

- ✅ Todos os pares são encontrados  
**ou**
- ⏰ O tempo limite se encerra
